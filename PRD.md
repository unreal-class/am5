# PRD — AM5 테니스 경기 관리 시스템

## 1. 개요

AM5 테니스 동호회의 일상적인 경기 운영을 디지털화하는 모바일 우선 웹 애플리케이션입니다. 출석 관리, 복식 대진 자동 생성, 경기 진행/결과 기록, 개인 전적 및 랭킹을 제공합니다.

## 2. 대상 사용자

- **일반 회원**: 출석, 본인 경기 확인/종료/결과 입력, 전적/랭킹 조회
- **관리자**: 회원 관리, 코트 대여 관리, 경기 기록 관리, 전체 모니터링, 테스트

## 3. 사용자 스토리

### 3.1 회원
- 회원가입 후 첫 로그인 시 비밀번호를 변경할 수 있다.
- 오늘 모임에 출석/퇴장할 수 있다.
- 본인이 배정된 경기를 확인할 수 있다.
- 본인이 참여 중인 경기를 종료하고 결과를 입력할 수 있다.
- 자신의 전적(승/패/승률)을 확인할 수 있다.
- 월간/연간/전체 랭킹을 조회할 수 있다.
- 자신의 정보(이름, 전화번호, 성별)를 수정할 수 있다.

### 3.2 관리자
- 모든 회원 정보를 조회/수정/삭제할 수 있다.
- 회원의 비밀번호를 초기화할 수 있다.
- 회원에게 관리자 권한을 부여/삭제할 수 있다.
- 게스트(비정규 회원)를 추가하고 승률을 설정할 수 있다.
- 코트의 대여 시작/종료를 관리할 수 있다.
- 경기 기록을 개별 또는 일괄 삭제할 수 있다.
- 코트별 진행 상황을 모니터링할 수 있다.
- 테스트 콘솔로 배정 로직을 시험할 수 있다.

## 5. 비기능 요구사항

### 5.1 UI/UX
- **모바일 우선**: 모든 화면이 360px~420px 너비에 최적화
- **한글 UI**: 모든 레이블, 버튼, 안내문이 한국어
- **큰 글씨**: 경기 카드(코트 번호, 선수명)와 재확인 대화상자는 강조를 위해 큰 폰트 사용
- **재확인**: 경기 종료, 퇴장, 로그아웃, 회원 삭제 등 위험 동작 시 커스텀 대화상자로 재확인

### 5.2 성능
- **폴링**: 6초 간격으로 경기/참석자 상태 자동 갱신
- **로딩**: 초기 데이터 로딩 중 스플래시 화면 표시
- **낙관적 업데이트**: 사용하지 않음 (모든 변경 후 Supabase 재조회)

### 5.3 보안
- **Row Level Security**: 모든 테이블에 RLS 적용
- **관리자 전용 API**: `/api/admin/*` 경로는 서버에서 관리자 권한 확인
- **세션 기반 인증**: Supabase Auth 세션 사용, 토큰 자동 갱신
- **비밀번호 정책**: 초기 비밀번호 `AM5AM5`, 첫 로그인 시 변경 강제

### 5.4 호환성
- 최신 Chrome, Safari, Firefox, Samsung Internet
- iOS Safari, Android Chrome
- 데스크톱 브라우저 기본 지원 (모바일에 최적화)

## 7. 배정 로직 상세

### 7.1 후보 선정
1. 오늘 모임에 출석 중(`checked_in_at` 존재, `checked_out_at` NULL)인 회원
2. 현재 진행 중인 경기가 없는 회원
3. 관리자 계정 제외

### 7.2 우선순위 정렬
```
오늘 경기 횟수(적을수록) → 대기 시간(길수록) → 출석 시간(빠를수록)
```

### 7.3 배정
상위 4명을 무조건 선택하여 A팀/B팀으로 분배. 조합 평가 점수:
```
genderPenalty(100000) + balanceGap(100) + partnerRepeat(1)
```
- genderPenalty: 여성 인원 있을 때 혼성 아닌 팀에 부과
- balanceGap: A팀/B팀 평균 승률 차이
- partnerRepeat: 이전에 같은 팀이었던 횟수

### 7.4 생성 조건
`min(가용 코트 수, ⌊대기 인원 ÷ 4⌋)` 만큼의 경기를 한 번에 생성.

## 8. 제약사항

- 코트는 3면 고정, 1/2/3번으로 식별
- 복식(4인 1경기)만 지원
- 무승부 불가 (점수 차이 필수)
- 동명이인 처리: 로그인 아이디에 구분값(`홍길동2`) 사용
- Supabase Auth의 이메일 기반 구조를 한글 ID로 우회 사용

## 9. 향후 계획 (비범위)

- 경기 중 무단 이탈 감지
- 푸시 알림 (경기 배정, 시작 알림)
- 경기 시간 통계 (평균 소요 시간 등)
- 출석률/참여율 통계
- 다국어 지원
- 다수 클럽 지원
